## Project Structure

```
rl_final/
│
├── main.py                  # Main streamlit, organize the pages
```

## Getting Started

### Prerequisites

- Python 3.8+
- Streamlit

### Installation

1. Clone the repository :
    ```bash
    git clone https://github.com/leez7onefl/GenAIComputerVisionLab4
    ```

2. Install dependencies into a virtual environment :
    ```bash
    built.bat
    ```

### Run Streamlit App

```bash
launch.bat
```
